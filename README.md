## 🌟 About DecalDen

**DecalDen** is an intuitive, streamlined web tool designed to bridge the gap between imagination and physical stickers. Think of it like ordering a pizza: upload your favorite artwork or vector, tweak your preferences, choose your material and finish, and hit send! 

Built with love and precision for creators, makers, and sticker enthusiasts alike. 

---

## ✨ Key Features

*   **🖼️ Smart Image Upload & Conversion:** Drop your artwork right in; the system processes and converts your design into a cut-ready decal format.
*   **📏 Dynamic Sizing & Thickness Adjustment:** Fine-tune border thicknesses, scaling, and dimensions with real-time UI previews.
*   **🎨 Material & Color Matrix:** Pick from an expansive lineup of premium vinyl colors, holographic sheens, and specialized finishes.
*   **⚡ Instant Pricing Calculator:** Transparent, real-time calculation based on surface area, cuts, and material specs.
*   **📦 Smooth Fulfillment Flow:** Seamless checkout workflow supporting free standard shipping and optional tracking upgrades.

## 🚀 Quick Start

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have a modern web browser and a local development server (like Live Server for VS Code) installed.

### Installation

```bash
git clone https://github.com/softbitestudio/decalden.git
cd decalden
```

# Open index.html or launch your local dev server
